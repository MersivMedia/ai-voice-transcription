## 🌐 Deployment

**Deploy to Vercel:**

1. Connect GitHub repository to Vercel
2. Set environment variable: `OPENAI_API_KEY` (optional - runs in demo mode without it)
3. Deploy automatically on git push

**Environment Variables:**
- `OPENAI_API_KEY` - Your OpenAI API key (optional, demo mode if not set)

## 🔧 Customization

### Ad Integration:
Replace placeholder ad slots with real ad code:
```html
<ins class="adsbygoogle" 
     style="display:block" 
     data-ad-client="ca-pub-YOUR-ID" 
     data-ad-slot="YOUR-SLOT-ID">
</ins>
```

### Theme Colors:
Modify CSS variables in the `:root` selector:
```css
:root {
    --primary: #1e40af;
    --accent: #3b82f6;
    /* Add your brand colors */
}
```

### AI Behavior:
Customize prompts in `api/simplify.js`:
- Adjust response format
- Modify analysis depth  
- Change tone and style
